## Add dependency

```kotlin
// settings.gradle.kts
dependencyResolutionManagement {
    repositories {
        google()
        mavenCentral()
        maven("https://jitpack.io")  // or your private registry
    }
}

// build.gradle.kts (app)
implementation("com.comission.shyware:shyware-sdk-android:1.0.0")
```

---

## Initialization

```kotlin
import com.comission.shyware.*

// Decode shyconfig.json
val config = Json.decodeFromString<ShyConfig>(shyconfigJson)

// Initialize receipt store (inject context)
val store = EncryptedReceiptStore(context, config.app.id)

// Initialize client (validates manifest — throws if invalid)
val client = VotingClient.from(config, store)
```

---

## Device attestation (required for receipt confirmation)

Provide Play Integrity signals before casting. The `cloudProjectNumber` is your Google Cloud project number (from Play Console → App integrity).

```kotlin
val integrity = PlayIntegrityProvider(context, cloudProjectNumber = 1234567890L)

// Request token with a server-issued nonce
val nonce = fetchNonceFromServer()    // your server endpoint
val (signals, token) = integrity.requestIntegrityToken(nonce)

// Forward token to your server for verdict verification
if (token != null) {
    val verdicts = verifyTokenOnServer(token)   // your server
    // If verdicts pass, server confirms; client already set signals
}

client.setRuntimeSignals(signals)
```

Check posture after setting signals:

```kotlin
val posture = client.effectivePosture()
if (posture.writeOnly) {
    showWriteOnlyUI(posture.fallbackReasons)
} else {
    showFullBallotUI()
}
```

See [Android Attestation](/sdk/android/attestation) for the full Play Integrity integration guide.

## Cast a ballot

```kotlin
val result = client.castBallot(
    pollId = "proposal-42",
    choice = "yes",
    input = IdentityInput.Didit(personId = diditPersonId)
)
// result.ballotId    — direction-free public identifier
// result.ballotNonce — private receipt material; do not expose
```

If posture is `recoverable`, the receipt is persisted to `EncryptedSharedPreferences` automatically.

---

## Read polls and tally

```kotlin
val polls = client.getAllPolls()
val poll  = client.getPoll("proposal-42")
val tally = client.getTally("proposal-42")
```

All network calls are suspend functions — call from a coroutine scope:

```kotlin
viewModelScope.launch {
    val polls = client.getAllPolls()
    _polls.value = polls
}
```

---

## Verify a receipt

```kotlin
val votes = client.getVotes("proposal-42")
val result = client.verifyReceipt(
    nonce = savedNonce,
    expectedChoice = "yes",
    votes = votes
)
// result.verified == true → ballot confirmed on-chain
```

Verification is offline — computes `SHA-256(nonce)` and scans the public vote list.
